## 优化：水人沙王根据技能有效范围寻找最远格子

目前水人施法是选择一个最远且能攻击到敌人的格子。但因为水人是远程单位，故在一些情况下会出现技能路径上没有敌方单位的情况导致空放技能。  

优化后则是让水人技能尽可能对更多的敌人造成伤害。
